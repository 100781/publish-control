Publish Control forces users to confirm publish to avoid accidental publishing of content.

== Description ==
It happens to everyone sooner or later. You accidentally click "Publish" before you are ready, and accidentally publish an incomplete piece of content.

When you click the "Publish" button after writing a piece of content, a small dialog box will appear to confirm if you really want to publish the content.

== Installation ==
1. Download plugin.
2. From the WordPress administration dashboard, go to Plugins > Add New > Upload.
3. Select file, click "Install Now."
4. Activate plugin.
5. By default, only "posts" are monitored by Publish Control. Navigate to settings to add post types.

OR

1. Download plugin.
2. Upload zipped plugin file to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. By default, only "posts" are monitored by Publish Control. Navigate to settings to add post types.
